Gridx Demo

Gridx is an incubation project as a basis for the next generation of Dojo Grid.

The key idea is to set up a highly extensible core and a rich set of modules to be loaded on demand, based on that appropriate modules can be picked up to consist different Grid instances to meet a wide range of requirements - MiniGrid(MobileGrid), AdvancedGrid, TreeGrid etc.

Check out the following demos and see how Gridx is more faster and flexible!

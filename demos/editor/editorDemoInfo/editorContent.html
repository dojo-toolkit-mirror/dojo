The dijit.Editor

The dijit.Editor is a powerful and flexible editor that is part of the Dojo Toolkit. This page is a fully-functional example of the Rich Text editor in action with numerous plugins enabled (61 plugins, to be exact). Feel free to experiment with the content in this pane to see how the various toolbar actions affect its content.
